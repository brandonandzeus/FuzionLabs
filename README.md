# FuzionLabs
Full custom clan for Monster Train

## Features
A new clan with

* 2 Champions
* 17 Units
* 22 Spells
* 12 Clan Artifacts (11 collectable and 1 starter)
* 1 Shop Upgrade
* 7 Status Effects + (5 Supporting Status Effects)
* 4 Triggers
* 5 Card Mechanics
* 1 new Resource type.
